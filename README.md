# recipes

## Recipe Website  

### Description  
This is a simple HTML-based recipe website. The project includes several recipe pages featuring easy-to-follow steps and ingredient lists.  

The goal is to practice HTML and Git fundamentals while creating a useful and structured recipe collection.  

### Features  
- Multiple recipe pages  
- Organized folder structure for images and HTML files  
- Clean and responsive HTML structure  
- Use of semantic HTML elements (`<ul>`, `<li>`, `<h1>`, `<p>`, `<img>`)
